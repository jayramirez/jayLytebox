Lytebox

Lytebox is a streamlined, flexible content viewer and dialog prompt. It can be used as an alert message, confirm message, popup dialog, or a simple message dialog which can be styled using html and css.

Example:

Let's get started

Files

We need 3 important files to make things work: jquery.js, lytebox.js and the lytebox.css. Simply embed the files in the head

		<head>
			<link href="dist/css/lytebox.css" rel="stylesheet">
			<script src="assets/js/jquery.js"></script>
			<script src="dist/js/lytebox.js"></script>
			...

Initialize with zero configuration

		<script>
		var lytebox = new Lytebox(); 
		</script>

Initialize with custom config

		<script>	
		var config  = {//config params
				escClose: true,
				transitionIn: 300,
				paddingTop: 100,
				bgClose : true
				};

		var lytebox = new Lytebox(config);
		</script>

Config Params (Initialization)

name type default description
top int null top position of the dialog. If the value is null, the dialog will be vertically aligned to the window.
paddingTop int 50 padding-top of the dialog container.
paddingBottom int 50 padding-bottom of the dialog container.
transitionIn int 100 transition speed of the lytebox while opening (fadeIn). The unit is miliseconds.
transitionOut int 100 transition speed of the lytebox while closig (fadeOut). The unit is miliseconds.
width int 360 dialog width
align string 'left' text alignment
okCaption string 'Ok' text caption for ok/confirm buttons.
cancelCaption string 'Cancel' text caption for cancel button.
btnAlign string 'right' button (ok, cancel) alignments for alert and confirm dialogs
close boolean true add close button for message dialog
escClose boolean false if set to true, lytebox can be closed by pressing esc key.
bgClose boolean true if set to true, lytebox can be closed by clicking the background of the dialog.

Methods

There are 3 main methods in this class: .dialog(), load(), .show();

.dialog(strMessage)

.dialog(ObjectParams)
.dialog(strMessage, ObjectParams)
.dialog(strMessage, strTitle)
.dialog(strMessage, strTitle, ObjectParams)

This method can be used as message, alert or confirm dialog. If the parameter is string, it will be automatically be considered as message dialog.

Demo:

Code:

//dialog call with string parameter.
lytebox.dialog('Hello world.')

//dialog call with object parameter.
lytebox.dialog({
	message: 'Hello world.', 
	title : 'Hi'
});

//the code above is the same as the code below
lytebox.dialog('Hello world.','Hi');

//dialog call as alert message.
lytebox.dialog({
	message: 'Hello world.', 
	title : 'Hi',
	type : 'alert'
});

//the code above is the same as the code below
lytebox.dialog('Hello world.', 'Hi', {type : 'alert'});


//dialog call as alert message with callback.
lytebox.dialog({
	message: 'Hello world.', 
	title : 'Hi',
	type : 'alert',
	onConfirm : function(){ 
			lytebox.dialog('You confirmed') 
		}
});

//dialog call as confirm message.
lytebox.dialog({
	message: 'Hello world.', 
	title : 'Hi',
	type : 'confirm'
});

//dialog call as confirm message with callback.
lytebox.dialog({
	message: 'Hello world.', 
	title : 'Hi',
	type : 'confirm',
	onConfirm : function(){ 
			lytebox.dialog('You confirmed') 
		},
	onCancel : function(){ 
			lytebox.dialog('You canceled') 
		}
});

ObjParams

name type default description
message string '' dialog message. can contain html tags.
title string null dialog title. can contain html tags.
align string Config.align Dialog content alignment.
close boolean Config.align add close button for message dialog
type string 'message' dialog types: message, alert, confirm
okCaption string Config.okCaption text caption for OK button.
cancelCaption string Config.cancelCaption text caption for cancel button.
btnAlign string Config.btnAling button alignment.
top int Config.top top position of the dialog. If the value is null, the dialog will be vertically aligned to the window.
width int Config.width dialog width
escClose boolean Config.escClose
lose the lytebox by pressing the esc key
bgClose boolean Config.bgClose Close the lytebox by clicking the background
onOpen function callback function after the lytebox showed.
onClose function callback function after the lytebox closed.
onConfirm function callback function after the clicking the confirm button.
onCancel function callback function after the clicking the cancel button.
